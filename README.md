# 항해 99 기술 블로그, Haelog
Haelog는 항해99 선원들을 위한 기술 블로그입니다.

## 구현기능
### 회원
닉네임, id, password
로그인 / 회원가입
조건과 맞는 부분과 맞지 않는 부분 표시
중복 확인. 

### 메인페이지
게시글 마크다운 미리보기
태그 기능 : 태그에 맞는 게시글만 보기

### 게시글
MD(Mark Down)로 게시글 작성
태그 

###댓글
댓글 CRUD
댓글 작성자만 수정, 삭제 버튼이 노출되도록 구현
